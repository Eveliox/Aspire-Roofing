# Images Directory

Place your images in this directory. The following images are referenced in the components:

## Required Images:

### Hero Section
- `hero-roof.jpg` - Full-width roof background image for hero section

### Why Aspire Section
- `why-aspire-roof.jpg` - Large roof image for the left column

### Local Specialists Section
- `roof-1.jpg` - First roof photo (top left)
- `roof-2.jpg` - Second roof photo (top right)
- `roof-3.jpg` - Third roof photo (bottom left)
- `roof-4.jpg` - Fourth roof photo (bottom right)

### About Section
- `about-bg.jpg` - Background image for about section

### Gallery Section
- `gallery-metal.jpg` - Metal roofing category image
- `gallery-tile.jpg` - Tile roofing category image
- `gallery-shingle.jpg` - Shingle roofing category image
- `gallery-flat.jpg` - Flat roofing category image

### Services Section
- `service-metal.jpg` - Metal roofing service image
- `service-tile.jpg` - Tile roofing service image
- `service-shingle.jpg` - Shingle roofing service image
- `service-flat.jpg` - Flat roofing service image

## Image Recommendations:
- Hero and About background images: 1920x1080px or larger
- Gallery and Service cards: 800x600px or similar aspect ratio
- Roof grid images: 600x600px or square format
- All images should be optimized for web (compressed, appropriate format)

The components will display placeholder text if images are missing, so you can add them gradually.
